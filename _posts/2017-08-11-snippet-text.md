# 1. Blurry Text

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .blur {
      color: transparent;
      text-shadow: 0 0 5px rgba(0,0,0,0.5);
    }
  </style>
</head>
<body>
  <p class="blur">This paragraph is styled with CSS.</p>
</body>
</html>
```
